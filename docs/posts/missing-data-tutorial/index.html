I’m working through the tutorial published by M. Fatih Tüzen at A Statistician’s R Notebook, and syndicated at R-Bloggers

Missing data are important, and dealing with them appropriately is critical to avoid the bias that might be introduced if they are ignored or improperly dealt with. Missing values decrease statistical power and compromise the validity of the analysis and conclusions.

Why missingness matters

Missing data can lead to multiple problems with analysis:

Biased results
if values are not missing at random, the analysis may systematically misrepresent the population
Reduced sample size
dropping missing values reduces data availability and compromises statistical strength
Model incompatibility
some modeling techniques in R (lm, glm) require complete data and will silently drop missing cases

BMI and Systolic BP

If we want to explore the relationship between BMI and Systolic BP, and there are missing values in the BMI data, we will end up with biased conclusions.

Show the code
# How many missing in BMI?
